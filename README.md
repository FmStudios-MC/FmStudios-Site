## üìñ About

This project contains the source code for the FmStudios landing page. The goal of the website is to provide a clean, accessible interface for users to discover our modpacks, view changelogs, and access download links (CurseForge/Modrinth).

### Key Features
* **Project Showcase:** detailed overviews of our modpacks.
* **Version Tracking:** Information on supported loaders (NeoForge, Fabric) and Minecraft versions.
* **Status Indicators:** Real-time updates on project status (e.g., Active, Coming Soon, Discontinued).
* **News & Updates:** The latest announcements regarding FmStudios projects.

## üöÄ Getting Started

To run a local copy of this website for development or testing purposes, follow these steps:

### Prerequisites
* A web browser (Chrome, Firefox, Edge, etc.)
* A code editor (VS Code recommended)
* Git installed on your machine

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/FmStudios-MC/FmStudios-Site.git](https://github.com/FmStudios-MC/FmStudios-Site.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd FmStudios-Site
    ```

3.  **Launch the site:**
    * Simply open the `index.html` file in your preferred browser.
    * *Optional:* If you are using VS Code, you can use the "Live Server" extension to launch a local development server.
